2. EHCo (Enhanced Huffman Coding with Run-Length and DCT-Based Audio Compression) (2025)
Featured / Short Description
EHCo is an academic research project focused on improving audio compression efficiency while maintaining high audio quality. The framework combines Discrete Cosine Transform (DCT), Run-Length Encoding (RLE), and Adaptive Huffman Coding (AHC) to achieve higher compression ratios with minimal distortion, making it suitable for efficient audio processing and real-time applications.

Role: Researcher & Developer

Outcome
Designed and evaluated a hybrid audio compression framework that achieved up to 8.45:1 compression ratio while maintaining PSNR > 40 dB and THD < 0.01%, demonstrating improved compression performance with preserved audio fidelity.

Tech Stack: Python • Signal Processing • DCT • RLE • Adaptive Huffman Coding

GitHub: https://github.com/punyeeta/ALGO2-RESEARCH.git
Researchers: Roxanne Locsin, Mark Vincent Limpahan, Rhenel Jhon Sajol

Detailed Description
EHCo is an academic research project developed to improve digital audio compression by balancing storage efficiency and audio quality. The framework integrates Discrete Cosine Transform (DCT) for signal optimization, Run-Length Encoding (RLE) for redundancy reduction, and Adaptive Huffman Coding (AHC) for efficient encoding.

The system was tested across audio samples ranging from 4 kHz to 48 kHz and demonstrated improved compression performance while maintaining high audio fidelity, making it applicable to streaming, telecommunications, and resource-efficient environments.
